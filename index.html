JTooltip

Not just another Tooltip library

by Jamie Tsao

Introduction

JTooltip is a lightweight, easy to use jQuery library for creating tooltips:

Hover over me! Don't forget me!

Simple to Use

You create tooltip(s) for any 'target' by calling JTooltip.create with a CSS selector, jQuery element, or DOM element.

JTooltip.create({ target : "#username", content: "Marshall Mathers" });
slimshady

Text from the title attribute can be used as content by not including the content option (use the contentAttr option to specify an attribute other than title).

<span class="jtooltip-me" title="Text from the 'title' attribute">Foo</span>
<span class="jtooltip-me" title="Allows for content re-use">Bar</span>

<script type="text/javascript">
    $(document).ready(function() {
        // multiple tooltips can be created by using CSS selectors
        JTooltip.create({ target : ".jtooltip-me" });
    });
</script>
